Persamaan Kuadrat

Persamaan Kuadrat

Pengertian Persamaan Kuadrat

Persamaan kuadrat adalah persamaan dengan derajat tertinggi dua. Bentuk umum dari persamaan kuadrat adalah:

ax² + bx + c = 0

Di mana a, b, dan c adalah konstanta, serta a ≠ 0. Nilai-nilai ini disebut koefisien.

Metode Penyelesaian

1. Faktorisasi

Metode ini digunakan jika persamaan dapat difaktorkan menjadi dua bentuk linear.

Contoh:

x² - 5x + 6 = 0

Faktor dari persamaan ini adalah:

(x - 2)(x - 3) = 0

Sehingga solusi dari x adalah:

x = 2 atau x = 3

2. Rumus Kuadrat

Jika faktorisasi tidak memungkinkan, kita dapat menggunakan rumus kuadrat:

x = [-b ± √(b² - 4ac)] / 2a

Contoh:

x² + 4x - 5 = 0

Dengan a = 1, b = 4, dan c = -5, maka:

x = [-4 ± √(4² - 4(1)(-5))] / 2(1)

x = [-4 ± √(16 + 20)] / 2

x = [-4 ± 6] / 2

Sehingga:

x = 1 atau x = -5

3. Melengkapkan Kuadrat

Metode ini melibatkan manipulasi aljabar untuk mengubah bentuk persamaan menjadi kuadrat sempurna.

Contoh:

x² + 6x + 5 = 0

Langkah-langkahnya adalah:

  1. Pindahkan konstanta: x² + 6x = -5
  2. Tambahkan ½(b)² ke kedua sisi: x² + 6x + 9 = 4
  3. Faktorkan sisi kiri: (x + 3)² = 4
  4. Selesaikan untuk x: x + 3 = ±2, sehingga x = -1 atau x = -5

Aplikasi Persamaan Kuadrat

Persamaan kuadrat banyak digunakan dalam berbagai bidang seperti fisika, ekonomi, teknik, dan statistik. Contohnya adalah menghitung lintasan benda yang dilempar, optimasi keuntungan, atau menganalisis fungsi profit dalam bisnis.

Pertidaksamaan Kuadrat

Pertidaksamaan Kuadrat

Pengertian Pertidaksamaan Kuadrat

Pertidaksamaan kuadrat melibatkan ekspresi kuadrat yang berkaitan dengan tanda ketidaksamaan seperti <, ≤, >, atau ≥. Pertidaksamaan ini sering digunakan dalam analisis interval dan solusi yang sesuai untuk nilai x dalam rentang tertentu.

Metode Penyelesaian

1. Membuat Grafik

Langkah ini melibatkan menggambar grafik parabola untuk menentukan kapan ekspresi kuadrat bernilai positif atau negatif.

2. Tes Nilai Interval

Contoh:

2x² - x - 1 > 0

  1. Faktorkan menjadi (2x - 1)(x + 1) > 0
  2. Tentukan akar-akar persamaan: x = ½ dan x = -1
  3. Cek tanda pada interval: (-∞, -1), (-1, ½), (½, ∞)

Contoh Soal

Temukan solusi dari pertidaksamaan: x² - 3x - 4 > 0

Penyelesaian:

  1. Faktorkan menjadi: (x - 4)(x + 1) > 0.
  2. Tentukan akar-akarnya: x = 4 dan x = -1.
  3. Analisis tanda pada interval:
    • Untuk x < -1, tanda positif.
    • Untuk -1 < x < 4, tanda negatif.
    • Untuk x > 4, tanda positif.
  4. Solusi: x ∈ (-∞, -1) ∪ (4, ∞).

Aplikasi Pertidaksamaan Kuadrat

Digunakan untuk menentukan area aman dalam rekayasa teknik, risiko dalam investasi, atau titik optimal dalam manajemen proyek.

Turunan

Turunan

Pengertian Turunan

Turunan adalah konsep kalkulus yang menentukan laju perubahan suatu fungsi terhadap variabel independennya. Turunan memberikan informasi tentang kemiringan kurva fungsi di suatu titik tertentu.

Aturan Turunan

1. Turunan Fungsi Polinomial

Jika f(x) = axⁿ, maka f'(x) = n * ax⁾ⁿ⁻¹.

2. Aturan Rantai

Jika f(x) = g(h(x)), maka f'(x) = g'(h(x)) * h'(x).

Contoh Soal.

1.Turunan Fungsi Polinomial

Diketahui fungsi f(x) = 3x3 - 5x2 + 2x - 7. Tentukan turunan pertama f'(x).

Penyelesaian

  1. Turunkan setiap suku fungsi:
    2. - Untuk 3x3: d/dx(3x3) = 9x2
- Untuk -5x2: d/dx(-5x2) = -10x
- Untuk 2x: d/dx(2x) = 2
- Untuk -7: d/dx(-7) = 0
  2. Gabungkan hasil turunan:
    3. f'(x) = 9x2 - 10x + 2

2. Turunan Fungsi Trigonometri

Diketahui fungsi g(x) = 2sin(x) + 3cos(x). Tentukan turunan pertama g'(x).

Penyelesaian

  1. Gunakan aturan turunan fungsi trigonometri:
    2. - d/dx(sin(x)) = cos(x)
- d/dx(cos(x)) = -sin(x)
  2. Turunkan setiap suku fungsi:
    3. - Untuk 2sin(x): d/dx(2sin(x)) = 2cos(x)
- Untuk 3cos(x): d/dx(3cos(x)) = -3sin(x)
  3. Gabungkan hasil turunan:
    4. g'(x) = 2cos(x) - 3sin(x)

Aplikasi Turunan

Turunan digunakan dalam optimasi (menentukan maksimum dan minimum), analisis fisik (kecepatan, percepatan), dan banyak bidang lainnya seperti ekonomi untuk analisis margin dan keuangan.
